Sayısal işlemler ile birlikte klasik adam asmaca deneyimi.
Kullanıcı önceden belirlenen gizli kelimenin harflerini tahmin ederek ortaya çıkarmaya çalışır.
Her yanlış tahminde kullanıcının canı azalır.
Kullanıcı kendi belirlediği sayısal işlemleri çözerek harf açabilir ve kelime hakkında ipucu alabilir.
